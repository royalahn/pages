## Overview

**AI HTML Hub** is a static documentation site that catalogs and showcases AI-generated HTML pages. It features a modern, searchable catalog interface where users can browse through various technical documents — from design guides and coding best practices to lecture summaries and workflow analyses.

## Live Demo

Visit the live site: **[AI HTML Hub](https://royalahn.github.io/pages/)**

## Features

- 🔍 **Real-time Search** — Instantly filter documents by title, description, or category
- 🌗 **Dark / Light Theme** — Seamless theme toggle with `localStorage` persistence
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop viewports
- ✨ **Modern UI** — Glassmorphism headers, gradient effects, scroll-reveal animations
- 📂 **Dynamic Catalog** — Pages are loaded from `pages.json` — no hardcoded HTML cards

## How It Works

1. **`index.html`** fetches `pages.json` on page load
2. Each entry in `pages.json` is rendered as an interactive card in a responsive grid
3. Clicking a card navigates to the corresponding HTML document in `ai-html/`
4. The search bar filters cards in real-time by title, description, and category

### Page Registry (`pages.json`)

Every document in `ai-html/` must be registered in `pages.json` with the following schema:

```json
{
  "title": "Page Title",
  "filename": "ai-html/example_page.html",
  "description": "A brief description of the page content.",
  "category": "Category Name",
  "date": "YYYY-MM-DD"
}
```

> **Note:** The `filename` field must include the `ai-html/` path prefix.

## Creating New Pages

1. **Copy** `template/base_template.html` as your starting point
2. **Modify** only the `<main>` section content
3. **Update** the header logo (`<svg>`) and title (`<h1>`) to match your page
4. **Save** the new file in the `ai-html/` directory
5. **Register** the page in `pages.json` with the `ai-html/` prefix in the filename

> ⚠️ Do **not** modify the Footer, Head settings, or other base structure. See [AGENTS.md](AGENTS.md) and [html_design_guide.md](template/html_design_guide.md) for detailed guidelines.

## Design System

The project follows a **"Modern Startup / SaaS Style"** design system featuring:

- **Glassmorphism** — Semi-transparent headers with `backdrop-blur`
- **Gradient Text** — Animated gradient highlights on key headings
- **Card Hover Effects** — Lift animation with shadow expansion on hover
- **Scroll Reveal** — Fade-in-up animation on scroll using `IntersectionObserver`
- **Glow Background** — Radial gradient glow effect for visual depth

Full design specifications are documented in [`template/html_design_guide.md`](template/html_design_guide.md).
