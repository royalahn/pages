## 개요

**AI HTML Hub**은 AI가 생성한 HTML 페이지들을 카탈로그 형태로 모아 보여주는 정적 문서 사이트입니다. 모던하고 검색 가능한 카탈로그 인터페이스를 통해 디자인 가이드, 코딩 모범 사례, 강의 요약, 워크플로우 분석 등 다양한 기술 문서를 탐색할 수 있습니다.

- 🔍 **실시간 검색** — 제목, 설명, 카테고리로 즉시 문서 필터링
- 🌗 **다크 / 라이트 테마** — `localStorage` 기반 테마 전환 및 상태 유지
- 📱 **완전 반응형** — 모바일, 태블릿, 데스크탑 뷰포트 최적화
- ✨ **모던 UI** — 글라스모피즘 헤더, 그라데이션 효과, 스크롤 리빌 애니메이션
- 📂 **동적 카탈로그** — `pages.json`에서 페이지 데이터를 로드하여 하드코딩 없이 카드 렌더링

## 동작 방식

1. **`index.html`**이 페이지 로드 시 `pages.json`을 fetch합니다
2. `pages.json`의 각 항목이 반응형 그리드에 인터랙티브 카드로 렌더링됩니다
3. 카드를 클릭하면 `ai-html/` 내의 해당 HTML 문서로 이동합니다
4. 검색바를 통해 제목, 설명, 카테고리별로 실시간 필터링이 가능합니다

### 페이지 레지스트리 (`pages.json`)

`ai-html/`에 추가하는 모든 문서는 반드시 `pages.json`에 아래 형식으로 등록해야 합니다:

```json
{
  "title": "페이지 제목",
  "filename": "ai-html/example_page.html",
  "description": "페이지 내용에 대한 간단한 설명.",
  "category": "카테고리명",
  "date": "YYYY-MM-DD"
}
```

> **참고:** `filename` 필드에는 반드시 `ai-html/` 경로 prefix를 포함해야 합니다.

## 새 페이지 만들기

1. `template/base_template.html`을 **복사**하여 시작합니다
2. `<main>` 섹션 내부만 **수정**합니다
3. 헤더의 로고(`<svg>`)와 타이틀(`<h1>`)을 페이지 성격에 맞게 **변경**합니다
4. 완성된 파일을 `ai-html/` 디렉토리에 **저장**합니다
5. `pages.json`에 `ai-html/` prefix를 포함하여 페이지 정보를 **등록**합니다

> ⚠️ Footer, Head 설정 등 기본 구조는 **절대 수정하지 마세요**. 상세 가이드라인은 [AGENTS.md](AGENTS.md)와 [html_design_guide.md](template/html_design_guide.md)를 참조하세요.

- **글로우 배경** — 시각적 깊이감을 위한 방사형 그라데이션 효과
